vim入门

发表于 2022-12-23 分类于 tools
本文字数： 623 阅读时长 ≈ 1 分钟

vim是Linux内置的一款文本编辑器，提供纯键盘编辑输入方式，脱离鼠标的同时又不输优雅。

​ 初学vim之前，需要了解它的三种模式：命令模式、输入模式、末行模式。其中，命令模式可以操作一些快捷键，例如：滚屏操作、光标移动。输入模式即常见的编辑模式，可以自由输入文本内容。末行模式提供了一些特殊的操作，例如：替换、快速查找、显示行号等。

​ 当使用vim打开一个文件时，默认为命令模式，在命令模式中按下i后，便进入了编辑模式；在编辑模式中按下Esc后，便又返回命令模式；在命令模式中按下Shift+;后，此时进入末行模式，在:后便可输入一些末行命令。

命令模式

  1. 注意区分大小写。
  2. 打开文件时默认进入命令模式，输入模式或者末行模式中按下Esc，进入命令模式。
控制光标

​ h：向左移动光标

​ j：向下移动光标

​ k：向上移动光标

​ l：向右移动光标

滚屏操作

​ Ctrl+u：向上滚动半屏

​ Ctrl+d：向下滚动半屏

​ Ctrl+b：向上滚动一屏

​ Ctrl+f：向下滚动一屏

移动到行首或行尾

​ 0：快速移动到光标所在行的行首

​ $：快速移动到光标所在行

​ ^：快速移动到光标所在行第一个非空字符

其他移动光标的操作

​ G：移动到文件最后行行首

​ 2+G：移动到指定行行首

​ gg：移动到文件第一行行首

​ >>：文本向右移动(缩进)

​ <<：文本向左移动

删除操作

​ x：删除光标后一个字符

​ X：删除光标前一个字符

​ dd：删除光标所在行

​ 10+dd：向下删除10行

​ d0：删除光标前所有内容

​ D ：删除光标后所有内容

​ daw：删除光标所在单词

可视模式

​ v：按字符移动光标选择内容

​ V：按行上下移动选择内容

撤销

​ u：撤销一步操作

​ Ctrl+r：反撤销

复制粘贴

配合可视模式，选中内容。

​ y：复制

​ p：粘贴

​ yy：复制光标所在行

重复操作

​ .：重复上一步操作

替换操作

​ r：如将n替换为m，先将光标移动到字母n下，然后再按r，再按下m即可。

​ R：按下R后，再按要替换的字母，替换后，光标会移动到后一个字符，可以再次执行替换操作。

输入模式

输入模式下，不能使用快捷键，可以正常输入内容。

进入输入模式

​ i：插入光标前一个字符

​ a：插入光标后一个字符

​ I：插入行首

​ A：插入行尾

​ o：向下另起一行，插入行首

​ O：向上另起一行，插入行首

末行模式

在命令模式下，按:进入末行模式。

退出

​ wq：保存并退出

​ q：直接退出

​ q!：强制退出

显示行号

​ set nu：显示行号

​ set nonu：关闭行号

查找

​ /查找内容：按n查找下一个，按N查找上一个

​ set hlsearch：高亮显示查找内容

替换

​ %s/abc/123/g：将当前文件中的所有abc替换成123

​ 1, 10s/abc/123/g：将第一行至第十行之间的 abc替换成123

​ %表示整个文件；1，10表示1到10行范围内

​ s表示替换

​ g表示全局
